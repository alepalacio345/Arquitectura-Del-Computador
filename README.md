# ⚙️ Arquitectura del Computador

¡Hola! 👋 Bienvenido a mi repositorio dedicado al estudio y la práctica de la Arquitectura del Computador. 

Aquí documento mi proceso de aprendizaje sobre cómo funciona el hardware a bajo nivel y cómo se comunica con el software, enfocado principalmente en la arquitectura y el lenguaje ensamblador **MIPS32**.

## 🛠️ Tecnologías y Herramientas
* **Lenguaje:** Ensamblador MIPS32.
* **Simulador:** MARS 4.5 (MIPS Assembler and Runtime Simulator) / QtSpim.
* **Teoría base:** *Estructura y diseño de computadores* (Patterson & Hennessy).

## 📂 Contenido del Repositorio
* **`/tareas`**: Apuntes y material de apoyo, incluyendo mi propia **Hoja de Referencia** en formato PDF (creada con LaTeX) con las instrucciones más usadas.
